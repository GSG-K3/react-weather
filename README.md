
# Description ( weather App using React):

  
## simple weather App that do the following:
- Default weather information
 - Add the city name and git the weather information and photo of this city 

 ## How to use our website !!:
  ### In terminal
 - clone these repo:
  ```  
   git clone https://github.com/GSG-K3/react-weather.git
   cd react-weather
 ```
- install node modules:
  ```
   npm i
   npm start
   ```
